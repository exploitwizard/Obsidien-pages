# Obsidian Pages

A Minecraft-themed login and registration system with Flask backend.

## Features
- User registration and login
- Secure password hashing
- Beautiful Minecraft background
- Responsive design
- PostgreSQL database

## Live Demo
- Frontend: https://[your-username].github.io/obsidian-pages
- Backend: https://obsidian-backend.onrender.com

## Local Development
1. Clone repository
2. Setup backend: `cd backend && pip install -r requirements.txt`
3. Run backend: `python app.py`
4. Open frontend: Open `frontend/index.html` in browser
